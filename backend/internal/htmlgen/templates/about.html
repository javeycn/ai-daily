关于 AI Daily

项目介绍

AI Daily 是一个自动化的 AI 资讯聚合平台。系统每天从全球多个权威 AI 新闻源自动采集最新资讯，通过大语言模型（LLM）智能生成中文摘要和分类标签，汇聚成每日精选 AI 日报。

数据来源

  • The Verge / VentureBeat / TechCrunch — 国际科技媒体
  • OpenAI / DeepMind / Google Research / Anthropic — AI 大厂官方博客
  • Unite.AI / THE DECODER / MarkTechPost / InfoQ AI — AI 专业媒体
  • Import AI / Simon Willison / Chip Huyen / Eugene Yan / Karpathy — 前沿思想领袖
  • Latent Space / The Gradient / The Batch / BAIR Blog — AI 深度分析
  • 36氪 / 机器之心 / IT之家 / 智东西 — 国内科技媒体

技术栈

后端采集Go + RSS/Atom 解析
数据存储SQLite (WAL 模式)
AI 摘要OpenAI GPT / Claude API
前端渲染Go html/template 静态生成
搜索Fuse.js 客户端搜索
部署静态站点 + Nginx

自动化流程

新闻采集→ 去重过滤→ LLM 摘要→ 分类标签→ 日报聚合→ 静态发布
